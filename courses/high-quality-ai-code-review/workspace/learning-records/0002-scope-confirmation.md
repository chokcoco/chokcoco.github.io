## 用户确认

- 主线受众为有研发经验的全栈工程师，兼顾 TL 和架构师。
- 默认会 Git 和基本 Prompt；MR、Agent、MCP、Harness 和简单指标从零解释。
- 第 1、9 章最低 7,000 有效字符，第 2–8 章最低 11,000，有余量但不能缩水。
- 课程用于公开售卖。
- 真实代码、接口名和类名可展示；数据库表名及组织、平台、域名、账号、人员全部脱敏。
- 指标章位于第 3 章。
- 实践最终脱离 `qms-server`，以思维启发为主；简化 AICR Harness 为可选支架，正文不过量展示代码。
- 每章交付 Markdown 和静态 HTML；不机械配置练习、答案和强验收栏目。
- 允许并要求联网补充公开论文、官方文档和工程博客。
- 使用中性技术视觉，可用 imagegen 生成封面。
- 用户在内容生成后完整 Review，再决定发布。
