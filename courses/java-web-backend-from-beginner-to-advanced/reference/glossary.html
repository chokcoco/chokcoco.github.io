Java Web 后端术语表

快速查阅样章中的核心术语和使用边界。

术语定义
JDKJava Development Kit，包含编译、运行、调试和诊断 Java 程序所需的工具。
JVMJava Virtual Machine，加载并执行 class 文件、管理运行时内存的抽象机器。
字节码Java 编译器写入 class 文件、由 JVM 解释或编译执行的指令表示。
引用能指向对象的值；引用变量保存的是引用值，不是对象本体。
形参方法声明中用于接收输入的局部变量。
实参调用方法时实际传入的表达式或值。
值传递调用方法时，把实参的值复制给形参；引用类型复制的也是引用值。
对象身份两个引用是否指向同一个对象，通常用 == 判断。
业务相等性两个对象在业务含义上是否相等，通常由 equals 定义。
参数校验按照接口预先声明的规则检查输入，并在不符合规则时拒绝继续处理。
约束对输入值的非空、长度、格式或范围等要求。
异常Java 用来表示非正常执行情况并沿调用栈传播的信息对象。
全局异常处理在 Web 边界集中接收异常，并转换为 HTTP 状态与响应体的机制。
Bean Validation通过约束注解描述输入规则，并由验证器检查对象或方法参数的规范。
错误契约API 对错误状态码、错误码、字段、消息和追踪信息的稳定约定。
Controller Advice跨多个 Spring MVC Controller 复用异常处理、绑定或模型逻辑的组件。
类加载JVM 从 class 字节定义类或接口，并进行链接和初始化的过程。
栈帧每次方法调用创建的数据结构，包含局部变量、操作数栈和返回信息。
堆JVM 线程共享的运行时数据区，类实例和数组从这里分配。
方法区JVMS 定义的线程共享逻辑区域，保存类结构等数据；HotSpot 的元空间是相关实现细节。
对象可达性从 GC Roots 等起点沿引用关系能否找到对象，是判断对象能否被回收的基础。
垃圾回收JVM 自动寻找不再可达的对象并回收相关存储空间的过程。
元空间HotSpot 用本地内存保存类元数据的实现区域，不等同于 Java 堆。
